📱 Installez l'appli sur votre téléphone

Ajoutez le concours sur votre écran d'accueil pour y accéder facilement, comme une vraie application.

iPhone — Safari
  1. Ouvrez le lien dans Safari
  2. Appuyez sur le bouton Partager □⇧
  3. Choisissez « Sur l'écran d'accueil »
  4. Appuyez sur « Ajouter »
Android — Chrome
  1. Ouvrez le lien dans Chrome
  2. Appuyez sur les 3 points ⋮ en haut à droite
  3. Choisissez « Ajouter à l'écran d'accueil »
  4. Appuyez sur « Installer »
